Principais Formatações

Negrito / Destaque

Para textos em negrito de forma semântica use a tag "STRONG".
também existe a não semântica que seria a tag "b", use apenas a "STRONG".

Itálico / Ênfáse

Para itálico semântica use a tag "EM".
com a forma não semântica use a tag "i"

Texto pequeno

Para texto pequeno use a tag SMALL.
ex: agora vamos ver um texto pequeno. quase não há diferença, mas ela existe

Texto Deletado

para um texto deletado use a tag DEL.
ex: aqui vai um texto deletado oefwoefhwoiefhwoifh

Texto Inserido (sublinhado)

para um texto sublinhado use a tag INS.
ex: aqui vai um texto inserido(sublinhado) isfubwuebfowefbow

Texto sobrescrito

use a tag SUP
ex:
sem estar com a tag: 3.x2
com a tag: 3.x2

Texto Subscrito

use a tag SUB.
ex: sem tag: H2O
com tag: H2O
