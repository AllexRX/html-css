Usando Links externos

Para adicionar links externos use a tag <a> e escreva o atributo href e adicione o valor url para esse atributo


        
            < a href="https://github.com/AllexRX"> Meu GitHub < /a >
        
        ↑ esse é o linha de código para por um link externo no seu site ↑
